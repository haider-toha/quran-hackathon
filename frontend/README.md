# Quran Hackathon — Frontend

Next.js 16 App Router frontend for the Quran-reading + RAG hackathon project.

## Stack

- Next.js 16 (App Router) + React 19
- TypeScript (strict)
- Tailwind v4
- pnpm

## Quickstart

```bash
pnpm install
pnpm dev          # starts on http://localhost:3000
```

## Scripts

```bash
pnpm lint         # eslint
pnpm type-check   # tsc --noEmit
pnpm format       # prettier --write
pnpm build        # next build
```

## Project conventions

Project-wide conventions live in the root `../CLAUDE.md`. The frontend section covers App Router rules, RSC defaults, TypeScript patterns, and anti-patterns. Coding-standards detail also lives in `../.claude/skills/frontend-coding-standards/`.
